Propiedade da Tabelas

table - Cria a tabela

tr - Cria cada linha

th - Titulo de cada Coluna

td - Informaçoes de cada linha

Nome Idade Peso Linguagem
Dimitri 30 85 html
João 34 95 css

Código html


                <table border="1">
                // Nome das Colunas
                <tr>
                <th>Nome</th>
                <th>Idade</th>
                <th>Peso</th>
                        <th>Linguagem</th>
                    </tr>
                    
                    // Linha 1        
                    <tr>
                    <td>Dimitri</td>
                    <td>30</td>
                    <td>85</td>
                    <td>html</td>
                    </tr>
                    
                    // Linha 2
                    <tr>
                    <td>João</td>
                    <td>34</td>
                    <td>95</td>
                    <td>css</td>
                    </tr>                
                    </table>

Código CSS


                    <style>                    
                    table {
                        width: 100%;
                        border: 1px solid black;    // Borda da tabela
                        border-collapse: collapse;  // Junta as bordas
                        text-align: center;         // Centraliza texto
                    }
                    
                    th, td {
                        border: 1px solid black;   // borda das células
                        padding: 5px;              // espaço interno
                    }
                    </style>
