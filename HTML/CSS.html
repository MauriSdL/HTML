Seletores e Classes

Seleciona o elemento html que voçê deseja estilizar.


                1 -- Seletor Universal - Todas as tags dentro do Body terão o mesmo comportamento.

Text-align

text-align:center; ==> Centraliza o texto

text-align:end; ==> Alinha conforme o fim da escrita (ex: direita em português).

text-align:justify; ==> Alinha o texto nos dois lados, como em jornais.

text-align:justify-all; ==> Justifica todas as linhas, inclusive a última.

text-align:left; ==> Alinha o texto à esquerda.

text-align:right; ==> Alinha o texto à direita.

text-align:start; ==> Alinha conforme o início da escrita (ex: esquerda em português).

text-align:inherit; ==> Copia o valor de text-align do elemento pai (explicitamente).

text-align:initial; ==> Redefine a propriedade para o valor padrão do CSS, ou seja, como se nenhum text-align tivesse sido aplicado

text-align:unset; ==> Se o elemento normalmente herda text-align (como o texto em <p> ou <div>), ele volta a herdar do pai.

text-align:match-parent; ==> Faz o texto herdar o alinhamento horizontal do elemento pai, mas de forma explícita, ou seja, o navegador calcula o alinhamento com base no valor real aplicado ao pai (por exemplo, left, right, center etc.).

Escrita de traz para frente

direction:rtl;

Define que o texto será exibido da direita para a esquerda (Right-To-Left), usado em idiomas como árabe e hebraico.


Inverte a palavra de traz para frente (usado junto com direction:rtl;)

unicode-bibi: bidi-override;

Força a direção definida em direction:rtl, ignorando a ordem normal dos caracteres.
Ou seja, ele inverte a ordem de exibição dos caracteres de acordo com a direção escolhida no direction:rtl.

Overflow


Controla o que acontece quando o conteúdo passa do tamanho da caixa (div).



Remover barra de rolagem da vertical.

overflow-y: hidden;



Remover barra de rolagem da horizontal.

overflow-x: hidden;



Parametros do Overflow


auto

Mostra barra do scroll, só se haver nescessidade de mostrar todo conteudo completo, ja que o conteudo é menor que o elemento pai (div)..

Esse auto, só aparece se for especificado uma autura para o elemento.



hidden

Mostra só o que cabe dentro do elemento pai (div), Corta (não mostra) o conteudo que passa para fora.



scroll

Mostra barra de rolagem para ver todo o conteudo que tem dentro, Mesmo o que nao é mostrado porque excede o limite de tamanho do elemento pai (div).



visible

Mostra o conteudo, Mesmo que ele saia para fora do elemento pai (div). (padrão)

Especificidade CS


Ordem de precedencia que o navegador prioriza executar:

1 - style inline
2 - #id
3 - tag.classe
4 - .class
5 - Tags

No caso de voçê multiplos seletores igual o ultimo será mostrado na tela.
h1{background:red;}
h1{background:green;}
h1{background:blue;} --> Esse será o mostrado na tela


Regra Important


! important ==> Ignora os parametros de precedencia mais forte executando esse que tem o important.


                #meuid {
                    color:red;
                }

                .minhaClasse {
                    color: blue;
                }

                h1 {
                    color: green; !important
                }

O certo seria primeiro o:
1-id
2-class
3-h1 sendo o ultimo
Mas usando a regra !important o h1 seria mostrado como se foce a prioridade
Ignorando a regra de prioridade dos outros.

Efeitos de Texto em card


white-space: normal;

Este texto quebra a linha normalmente quando chega ao final da caixa.
white-space: nowrap;

Este texto não quebra a linha e continua na mesma linha mesmo que ultrapasse a borda da caixa.
text-overflow: ellipsis;

Este texto é muito longo e vai mostrar reticências...
height: auto;

A altura desta caixa se ajusta automaticamente ao tamanho do texto.
overflow: hidden;

Este texto é muito longo e vai ser cortado porque a caixa é pequena.
