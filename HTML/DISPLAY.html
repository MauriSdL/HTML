none

O elemento não aparece na página (é como se não existisse).


contents

Faz o elemento sumir visualmente, mas mantém seus filhos no fluxo da página.

Tags que não sofrem alteração de display (renderização neutra)

São tags sem impacto visual
O navegador as interpreta como metadados ou estrutura
Não aparecem na página.


            

                < head >
                < meta >
                < link >
                < title >
                < style >
                < script >
                < base >
                < param >
                < source >
                < track >

inline-block

Mistura os dois: fica na mesma linha (como inline), mas permite definir largura e altura (como block).
