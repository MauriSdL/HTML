Ajuste automático de colunas no Grid

grid-template-columns:repeat(autofix, minmax(200px. 1fr));

Extender comprimento da coluna

Isso faz com que o Grid crie várias colunas automaticamente, cada uma tendo pelo menos 200px, mas podendo aumentar até ocupar o espaço disponível.

O auto-fit empurra as colunas para se ajustar ao espaço da tela, fazendo o layout ser responsivo.

Em outras palavras: ele monta quantas colunas couberem, cada uma crescendo e se encolhendo de acordo com o tamanho da tela.

grid-column:1/5; -- Da primeira ate a quinta coluna.


Preenchimento automático de colunas

Isso faz o Grid criar o maior número possível de colunas com largura mínima de 200px, e se tiver espaço sobrando, cada coluna se expande até preencher o restante da área.

O auto-fill tenta preencher a linha com o máximo de colunas, mesmo que sobrem espaços vazios.

grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));


Grid-template-areas

Nomeando áreas -- Você define a estrutura da sua grade usando a grid-template-areaspropriedade no contêiner da grade. Essa propriedade usa uma sequência de nomes para representar visualmente o layout da grade, como um mapa

Atribuição de elementos -- Em seguida, você atribui elementos HTML individuais a essas áreas nomeadas usando a grid-areapropriedade nos elementos filhos.

Criando espaço vazio -- Você pode deixar células vazias na grid-template-areasdefinição usando um ponto final ( .).

ormato retangular -- As áreas da grade devem ser retangulares; não é possível criar áreas em forma de T ou L. 


                .grid-container {
                    display: grid;
                    grid-template-columns: 1fr 1fr;
                    grid-template-rows: 50px 1fr 30px;
                    grid-template-areas:
                        "header header"
                        "sidebar main"
                        "footer footer";
                }

                .header {
                    grid-area: header;
                }

                .sidebar {
                    grid-area: sidebar;
                }

                .main {
                    grid-area: main;
                }

                .footer {
                    grid-area: footer;
                }
